### What I Do

- Build fullstack web apps with `React`, `Vue`, `Nuxt`, `TypeScript`, `Node.js` and modern tooling.
- Design and ship product UI with strong UX, responsive layouts, readable state handling and clean component architecture.
- Integrate authentication, databases, realtime features, deployment pipelines and production-ready hosting.
- Work with Firebase, Supabase/Postgres, Prisma, Docker, PM2, GitHub Actions and static/cloud deployment flows.
- Improve older projects: migrate stacks, clean architecture, fix CI/CD, document setup and make apps deployable.
